# Trinus ICP Profiles

## Company Profile

**Trinus Corporation** sells IT consulting, data analytics, cloud engineering, managed services, and staffing to mid-market and enterprise companies. Their sweet spot: companies with complex data environments that lack the internal capacity or specialized expertise to manage them.

**Proof of enterprise reach:** Amgen, AT&T, Bank of America, Cisco, Johnson & Johnson, Intel, Intuit, KPMG, Qualcomm, Siemens, Verizon, VMware.

---

| Industry | Why Trinus Wins Here | Primary Pain |

## Buying Context by Service Line

### BI & Analytics
**Buyers:** CDO, VP Analytics, Director BI
**Trigger:** "We have data but can't get answers fast enough" / fragmented tools / new CDO mandate
**Trinus offer:** Snowflake + Tableau/Power BI implementation, data warehouse modernization, self-service analytics buildout

### Data Management
**Buyers:** CDO, VP Data Engineering, Data Governance Lead
**Trigger:** Post-merger data integration / compliance audit / data quality failures in production
**Trinus offer:** MDM implementation, data catalog, governance framework, data quality tooling

### Cloud Engineering
**Buyers:** CTO, VP Engineering, Cloud Architect
**Trigger:** Announced cloud migration / expiring on-premise contracts / new cloud-native product initiative
**Trinus offer:** AWS/Azure/GCP architecture, migration execution, cloud-native data platform

### IT Consulting
**Buyers:** CIO, IT Director
**Trigger:** Strategic IT roadmap reset / new CIO hire / board-mandated cost reduction
**Trinus offer:** IT strategy, vendor selection, architecture review, program management

### Staffing
**Buyers:** IT Director, Engineering Manager, CHRO
**Trigger:** Hiring spike for data engineers, cloud architects, BI developers / talent shortage
**Trinus offer:** Contract, contract-to-hire, direct placement of data and cloud talent

### Managed Services
**Buyers:** CIO, IT Director, VP Operations
**Trigger:** Team stretched thin / want to offload operational burden / cost pressure on headcount
**Trinus offer:** 24/7 managed analytics, cloud operations, data platform management

---

## Disqualify If:

- Company headcount < 200 (too small for enterprise engagement model)
- No in-house IT or data function (no internal champion possible)
- Consumer software / B2C product companies without data infrastructure needs
- Startup with no data maturity (no budget, no complexity to solve)
