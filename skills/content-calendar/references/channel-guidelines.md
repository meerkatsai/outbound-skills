# Channel-Specific Content Guidelines

## LinkedIn Posts

### Format
- **Type**: Text post (no articles — native text posts get 5–10x more reach)
- **Length**: 800–1,300 characters
- **Structure**: Hook (first 2 lines before "see more") → Body → CTA
- **Frequency**: 1 post per day

### Hook Patterns
Use one per post:
1. **Contrarian**: "Most agencies think scaling means hiring. It doesn't."
2. **Specific number**: "We onboarded 3 new clients last week without adding headcount."
3. **Question**: "What if your best playbook could run itself across every client?"
4. **Before/after**: "6 months ago: 4 hours per client setup. Now: 15 minutes."
5. **Pattern interrupt**: "Stop rebuilding the same campaign from scratch for every client."

### Content Pillars
- Pain point empathy — name the struggle without immediately pitching
- How-we-do-it — show the workflow, not just the result
- Industry commentary — react to trends/news with a relevant take
- Social proof — results and outcomes (when available)
- Educational micro-content — one actionable tip per post

### CTA Types
- Soft: "What's your approach?" / "Agree or disagree?"
- Medium: "DM me if you want to see how" / "Link in comments"
- Hard: "Book a demo" (max 1 per week)

### Rules
- Post 8–10 AM in target audience timezone
- Engage with comments within first hour
- Never start with "I'm excited to announce" or "Thrilled to share"
- Use line breaks generously — walls of text kill engagement
- No hashtags in body; add 3–5 at very end if desired

## YouTube Videos

### Format
- **Type**: Educational tutorials, walkthroughs, thought leadership
- **Length**: 8–15 minutes
- **Structure**: Hook (0–30s) → Problem (30s–2min) → Solution (2–10min) → CTA (last 1–2min)
- **Frequency**: 1 per content cycle

### Video Types
1. **Screen-share tutorial**: "How to [do X] in [Y] minutes" — show the product
2. **Talking head + screen share**: Problem statement → solution walkthrough
3. **Comparison**: "[Tool A] vs [Tool B] — Which is best for [use case]?"
4. **News reaction**: React to industry news with a relevant perspective

### Title Rules
- Lead with search query or pain point, not product name
- Include current year for comparison/tools content
- Under 60 characters
- Use numbers: "5 Ways to...", "In Under 10 Minutes"

### Thumbnail
- High contrast text (3–5 words max)
- Face showing emotion if talking head
- Before/after or versus layout for comparisons
- No product screenshots — too small to read

### Description Template
```
[2–3 sentence summary]

Timestamps:
0:00 — [Topic]
2:15 — [Topic]

Resources:
- [Link 1]

Try [Product] free: [link]
Book a demo: [calendly link]
```

### CTAs
- Mid-roll: "If this is helpful, subscribe — new tutorial every week"
- End: "Try [Product] free — link in description"
- Pinned comment: Link to most relevant resource

## Blog Posts (SEO)

### Format
- **Type**: Long-form SEO article
- **Length**: 1,500–2,500 words
- **Frequency**: 2–3 per content cycle

### Structure Template
```
# [SEO Title — lead with search query]

[Definition block: 2–3 sentences answering the core question.
This is what AI assistants and featured snippets pull.]

## [What — explain the concept]
## [Why it matters — connect to ICP pain points]
## [How to do it — step-by-step]
## [Product angle — natural, not a pitch]
## Key Takeaways (3–5 bullets)
## Next Steps (soft CTA)
```

### Title Rules
- Lead with query phrasing: "How to...", "Best...", "What is..."
- Don't stuff ICP terms ("for agencies") when search query is broader
- Include year for comparison/tools/trends content
- Meta title: 50–60 characters

### Definition Block
Every post starts with 2–3 sentences directly answering the query. Critical for:
- AI assistants (ChatGPT, Perplexity, Claude) pulling from early paragraphs
- Google featured snippets
- Immediately satisfying search intent

### SEO Checklist
- [ ] Title includes primary search query
- [ ] Definition block in first paragraph
- [ ] H2s use related search queries
- [ ] At least one image or diagram
- [ ] Meta description (150–160 chars)
- [ ] Internal links to 2+ related pages
- [ ] Single clear CTA at end
